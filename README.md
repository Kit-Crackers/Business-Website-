# Sweet Treat Bakery — Website Project

Welcome to **Sweet Treat Bakery**, an elegant and user-friendly website built to showcase bakery products and provide customers with a delightful online experience.  
This project is designed as a mini-project for academic purposes and demonstrates skills in **HTML, CSS, JavaScript (front-end)** without using any backend technology.

## Features

✔ Responsive homepage with banner / carousel  
✔ Bakery product gallery with images & description  
✔ Add-to-Wishlist system  
✔ Search bar to find bakery items easily  
✔ Contact page to message the bakery owner via WhatsApp  
✔ About page describing bakery story and mission  
✔ Order customization option  
✔ Location & delivery limited to one city  
✔ Attractive white & violet theme with modern layout  

## Technologies Used

| Technology | Purpose |
|-----------|---------|
| HTML5 | Structure of the website |
| CSS3 | Styling & responsive layout |
| JavaScript | Interactivity (wishlist, order message) |
| WhatsApp API | For customer message/order support |
